# In-App Changelog Copy

## Version 1.0.0

### English

AXPO 1.0 is the first full release of the app.

Create and manage simulations in one place.
Share simulations with clients in a clear, easy-to-read format.
Review pricing, results, and simulation details quickly.
Organize users, agencies, and customer information.
Keep track of important activity and updates.
Use built-in reports, analytics, and customizable templates.

### Español

AXPO 1.0 es la primera versión completa de la aplicación.

Crear y gestionar simulaciones desde un solo lugar.
Compartir simulaciones con clientes de forma clara y fácil de leer.
Revisar rápidamente precios, resultados y detalles de cada simulación.
Organizar usuarios, agencias e información de clientes.
Hacer seguimiento de actividad y novedades importantes.
Utilizar informes, analíticas y plantillas personalizables.
